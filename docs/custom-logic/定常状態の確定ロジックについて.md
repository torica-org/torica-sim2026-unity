# 定常状態の確定ロジックについて

## 以前のロジック

### 定常確定中

1. 前後荷重の理論値をそれぞれ算出
2. 理論値に一致する係数を算出

### フライト中

1. 全備の重心位置を算出
2. 係数をかけた補正後のパイロット重心位置を算出
3. 係数をかける前の補正前パイロット重心位置を算出

## 現在のロジック

### 定常確定中

1. 重心センサーの生の値からパイロットの重心位置 `pilotCenterOfGRaw` を算出
  ```cs
   // 重心フレーム上での桁中心モーメントについて，（前後センサにかかる荷重によるモーメント）＝（パイロットの体重によるモーメント）とし，その両辺をパイロットの体重で割った式
  pilotCenterOfGRaw = (massRightNow * lengthForward + massBackwardRightNow * lengthBackward) / pilotMassReal;
  ```
2. パイロットの定常における重心位置の理論値 `pilotCenterOfGTheoretical` を算出
  ```cs
  // シミュレーター上での桁中心モーメントについて，（パイロットの体重によるモーメント）＝（空虚重量〈パイロットなしの機体重量〉によるモーメント）とし，その両辺をパイロットの体重で割った式
  float pilotCenterOfGTheoretical = (-1 * aircraftMass * aircraftCenterOfMass) / pilotMassReal;
  ```
  > [!NOTE]
  > 定常状態では「空虚重量（パイロットなしの機体重量）によるモーメント」と「パイロットの体重によるモーメント」が等しくなることを利用
3. 補正値 `pilotCenterOfGOffset` を算出
  ```cs
  // 補正値の算出
  pilotCenterOfGOffset = pilotCenterOfGTheoretical - pilotCenterOfGRaw;
  ```

### フライト中

1. 重心センサーの生の値から得られたパイロットの重心位置 `pilotCenterOfGRaw` を算出
  ```cs
  // 重心フレーム上での桁中心モーメントについて，（前後センサにかかる荷重によるモーメント）＝（パイロットの体重によるモーメント）とし，その両辺をパイロットの体重で割った式
  pilotCenterOfGRaw = (massRightNow * lengthForward + massBackwardRightNow * lengthBackward) / pilotMassReal;
  ```
2. 補正後のパイロット重心位置 `pilotCenterOfG` を算出
  ```cs
  // 補正
  pilotCenterOfG = pilotCenterOfGRaw + pilotCenterOfGOffset;
  ```
3. 全備の重心位置 `centerOfG` を算出
  ```cs
  // 桁中心モーメントについて，（パイロット体重と空虚重量〈パイロットなしの機体重量〉によるモーメント）＝（全備重量によるモーメント）とし，その両辺を全備重量で割った式
  centerOfG = (pilotMass * pilotCenterOfG + aircraftMass * aircraftCenterOfMass) / (pilotMass + aircraftMass);
  ```
